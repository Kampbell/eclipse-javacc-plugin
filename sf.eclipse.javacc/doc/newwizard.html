Templates for JavaCC file or JJTree file

The New Wizard can generate for you a basic .jj or .jjt file.
To launch the Wizard click on "New" -> "Other..." or Ctrl+N :


The files used as templates are in the folder "/templates" of the plugin.
There you will find two files taken from the examples given in the JavaCC distribution.
When you check the "JJ Tree file" you just choose "new_file.jjt" instead of "new_file.jj".
The choosen file is copied where you indicate in "Container:" with the name you give in "File name:"
