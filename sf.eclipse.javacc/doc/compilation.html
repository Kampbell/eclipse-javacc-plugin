Compilation with JavaCC, JJTree and JJDoc

When you edit a .jj file or a .jjt file you see two icons on the toolbar :

The "JJC" icon stands for JavaCC compile if you are editing a .jj file
and for JJTree if you are editing à .jjt file.

The "JJD" icon stands for JJDoc.

Note that this icon is the only way to call JJDoc.

For JavaCC or JJTree you can also select the file in the Navigator or the Package Explorer,
right clic to open context menu and choose "Compile with JavaCC".
